Installation Options

Step 1:

Launch ARbs

Install ObsidianOS directly from an existing Arch Linux system using our installer script.


Just download this page's URL and run it with bash

Step 1.1:

Quickly launch ARbs

If you are launching this often, use the following line:

sudo bash -c "$(curl -fsSL https://ba.sh/ARbs)"

Step 2:

Choose your settings

Configure the soon to be installed with an sfs/mkobsfs file and choose your partitioning scheme

Hit the big install button and start!.
